O que é CSS?

Introdução

CSS (Cascading Style Sheets) é uma linguagem usada para descrever a apresentação de um documento HTML. Enquanto o HTML estrutura o conteúdo de uma página, o CSS é responsável por definir como esse conteúdo será exibido em termos de cores, fontes, layout e outros aspectos visuais.

O que o CSS faz?

O CSS permite que você separe a estrutura do conteúdo (HTML) da sua apresentação (estilos visuais). Com o CSS, é possível:

  • Alterar as cores de fundo e de texto.
  • Definir o tamanho e o tipo das fontes.
  • Organizar o layout da página, controlando margens, espaçamento e alinhamento.
  • Adicionar efeitos de transição e animações.
  • Criar designs responsivos, adaptando o layout para diferentes tamanhos de tela.

Exemplo de uso do CSS

Um exemplo simples de HTML com CSS seria:

<html>
    <head>
        <title>Minha Página com Estilo</title>
        <style>
            body {
                background-color: #f0f0f0;
                font-family: Arial, sans-serif;
            }
            h1 {
                color: #4CAF50;
            }
            p {
                color: #555;
            }
        </style>
    </head>
    <body>
        <h1>Bem-vindo à minha página estilizada!</h1>
        <p>Este é um parágrafo com estilo aplicado usando CSS.</p>
    </body>
</html>

Principais Propriedades do CSS

Algumas das propriedades mais comuns do CSS incluem:

  • color: Define a cor do texto.
  • background-color: Define a cor de fundo do elemento.
  • font-family: Define a fonte do texto.
  • font-size: Define o tamanho da fonte.
  • margin: Define o espaço externo de um elemento.
  • padding: Define o espaço interno de um elemento.
  • width e height: Controlam o tamanho de um elemento.
